# Welcome to Onix react example project

Here you can find boilerplate for your React project.

## Getting started
To run the project locally, you need:
- [x] install dependencies `npm install`
- [x] run the project `npm start`

## Available scripts
- [x] `npm start` - run the project locally
- [x] `npm run build` - build the project
- [x] `npm run storybook` - run storybook
- [x] `npm run build-storybook` - build storybook
- [x] `npm run test` - run unit tests
- [x] `npm run lint` - run eslint
- [x] `npm run lint:fix` - run eslint with fix

## List of features
- [x] Pure react application
- [x] Redux store management with persist storage
- [x] Multi-language support
- [x] Storybook component description
- [x] Eslint with airbnb config
- [x] Pre-commit and pre-push hooks
- [x] Sonarqube integration
- [x] React testing library for unit testing
